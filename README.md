# what is this?

Scaffold for Flask CRUD apps

# how to use?

* __dependencies__: `poetry install`
* __env var__: `ln -sf .env.dev .env`
* __scaffold db__: `make seed`
* __run__: `make flask`
* __everything else__: `make help`
